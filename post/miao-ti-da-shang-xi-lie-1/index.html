（1）证明：由①知显然x != 0
∴y - x = 1/x即y=x + 1/x，由基本不等式得证
（2）解：由②得y^2 + ax + b + a/x = 0
y + ay + b = 0;
由韦达知y1+y2 = -a,y1y2 = b
a^2 最小值当y1y2取±2时，b^2最小值为0，此时y1+y2 = 0
而y1=-2,y2=2时两者均取最小值
此时有最小值16
